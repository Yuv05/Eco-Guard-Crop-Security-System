# EcoGuard: Safe & Eco-Friendly Crop and Livestock Security System

## Project Description
EcoGuard is an IoT-based agricultural security system designed to protect crops and livestock safely without harming animals or humans.

## Components Used
- Arduino Nano
- PIR Sensor
- Ultrasonic Sensor
- Rain Sensor
- GSM SIM800L Module
- Relay Module
- LED / Alarm

## Working Principle
- PIR detects motion
- Ultrasonic verifies intruder distance
- Relay activates light/alarm
- GSM sends SMS alert to farmer

## Applications
- Crop protection
- Livestock security
- Smart agriculture
